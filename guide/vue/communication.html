    # Vue的通信方式

    组件间通信常用方式

    • props

      父组件给子组件传值

      // child
props: { msg: String }
// parent
<HelloWorld msg="Welcome to Your Vue.js App"/>
    • event

      子组件给父组件传值

      <template>
// parent
    <Cart @add="cartAdd($event)"></Cart>
</template>
<script>
// child
    this.$emit('add', good)
</script>
    • vuex

      创建唯⼀的全局数据管理者store，通过它管理数据并通知组件状态变更。

    • EventBus

      事件总线：任意两个组件之间传值常⽤事件总线 或 vuex的⽅式。（如果你在某一个页面刷新了之后，与之相关的EventBus会被移除，这样就导致业务走不下去。还要就是如果业务有反复操作的页面，EventBus 在监听的时候就会触发很多次，也是一个非常大的隐患。这时候我们就需要好好处理 EventBus 在项目中的关系。通常会用到，在vue页面销毁时，同时移除EventBus 事件监听）

    边界情况

    • $parent

      兄弟组件之间通信可通过共同祖辈搭桥，$parent或$root。

      // brother1
this.$parent.$on('foo', handle)
// brother2
this.$parent.$emit('foo')
this.$root.$emit('foo')
    • $children

      ⽗组件可以通过$children访问⼦组件实现⽗⼦通信。

      注意：$children不能保证⼦元素顺序
      // parent
this.$children[0].xx = 'xxx'
    • $root
    • $refs

      获取⼦节点引⽤

      <template>
// parent
<HelloWorld ref="hw"/>
</template>
<script>
    mounted() {
        this.$refs.hw.xx = 'xxx'
    }
</script>
    • provide/inject

      能够实现祖先和后代之间传值,父组件或者不管多深的组件都可以获取

      // ancestor
provide() {
    return {foo: 'foo'}
}
// descendant
inject: ['foo']

    ⾮prop特性

    • $attrs
    • $listeners

    包含了⽗作⽤域中不作为 prop 被识别 (且获取) 的特性绑定 ( class 和 style 除外)。当⼀个组件没有 声明任何 prop 时，这⾥会包含所有⽗作⽤域的绑定 ( class 和 style 除外)，并且可以通过 v- bind="$attrs" 传⼊内部组件——在创建⾼级别的组件时⾮常有⽤。

    插槽

    插槽语法是Vue 实现的内容分发 API，⽤于复合组件开发。该技术在通⽤组件库开发中有⼤量应⽤。

    • 匿名插槽 
      <template>
// comp1
    <div>
        <slot></slot>
    </div>
// parent
    <comp>hello</comp>
</template>
    • 具名插槽

      将内容分发到⼦组件指定位置

      <template>
// comp2
    <div>
        <slot></slot>
        <slot name="content"></slot>
    </div>
// parent
    <Comp2>
<!-- 默认插槽⽤default做参数 -->
        <template v-slot:default>具名插槽</template>
<!-- 具名插槽⽤插槽名做参数 -->
        <template v-slot:content>内容...</template>
    </Comp2>
</template>
    • 作用域插槽

      分发内容要⽤到⼦组件中的数据

      <template>
// comp3
    <div>
       <slot :foo="foo"></slot>
    </div>
// parent
    <Comp3>
<!-- 把v-slot的值指定为作⽤域上下⽂对象 -->
        <template v-slot:default="slotProps">来⾃⼦组件数据：{{slotProps.foo}}</template>
    </Comp3>
</template>
